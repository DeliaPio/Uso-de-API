En la aplicación se hace una petición a la API publica POKEAPI por medio de  un GET, hacemos uso de este GET ya que lo único que estamos haciendo es recibiendo información existente en esta API.

La información requerida para darnos una respuesta de retorno es un número entre 1 y 898, que es recibida por un input y es agregada a la URL de nuestro FETCH, trayendo y mostrando de esta forma la información del orden, nombre y tipo de pokemon al que hace referencia el número que fue enviado por nuestro servidor.
